# Rafraîchir le fil

Vous pouvez mettre en file d’attente un rafraîchissement manuel du fil d'actualité Braze à tout moment en appelant :

```java
Braze.requestFeedRefresh()
```

Consultez notre [KDoc][16] pour plus d’informations.
